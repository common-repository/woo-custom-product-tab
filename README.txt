== Description ==
Woo Custom Product Tab plugin allows site admin to add custom tabs to products. 

= Plugin Functionality: =
* Allow user to create custom product tab.
* Allow user to chose created product tabs for particular product.
* Display chosen custom tabs on product detail page.

== Installation ==
* Download the plugin
* Upload the folder "woo-custom-product-tab" to wp-content/plugins (or upload a zip through the WordPress admin)

== Frequently Asked Questions ==
= How to Setup : =
* Step-1: Go to edit product section in product data section you will find 'Custom Product Tab'.
* Step-2: Select tabs which you want to set for that product.
* Step-3: Save product.
